## ‚ö° Installation

### Quick Start

Launch Simplify11 from PowerShell:

```powershell
iwr "https://dub.sh/simplify11" | iex
```

### Full Installation

Creates a shortcut in the Start Menu to launch the latest version:

```powershell
iwr "https://dub.sh/s11install" | iex
```
